# IPASS 자체제작 비공식 이벤트 출품 앱
2020년 아이패스 본캐부캐 챌린지에 출품한 아이패스 비공식 앱입니다. 저의 본캐는 학생이며 저의 부캐는 개발자&디자이너다 라는 마음으로 개발부터 디자인까지 기획했습니다.
비록 학업으로 미완성으로 끝냈지만, 제대로 무엇 하나를 기획해서 만들어본다는 경험이 뜻깊었습니다.
